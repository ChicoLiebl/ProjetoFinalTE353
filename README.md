# Projeto Final TE353
Projeto Final da Disciplina TE353 - Programação Orientada a Objetos

## Objetivo

  Desenvolver um programa em C++ utilizando os seguintes recursos:

- Programação orientada a Objetos

- Sobrecarga de Operadores

- Herança Simples

- Herança Múltipla

- Funções Virtuais

- Polimorfismo

- Interface amigável

- Diagrama de Classes
